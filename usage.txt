Usage: g [OPTIONS] COMMAND [ARGS]

Commands:

  clone URL    clone repo into a known location
  sh GLOB      spawn a shell rooted in the project
  create NAME  create a project of NAME on github
  list [GLOB]  list projects (optionally matching GLOB)
  resume       interactive select recently used

Environment:

  G_GITHUB_TOKEN    your github personal access token
  G_GITHUB_USER     your github user name
  G_PROJECT_ROOT    root for cloning projects (default ~/Projects)
